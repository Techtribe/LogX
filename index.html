LogX

A lightweight logging setup that helps you with local and remote (RTMFP) debugging.

Download this project as a .zip file Download this project as a tar.gz file

A lightweight logging setup that helps you with local and remote debugging in Actionscript 3. It is made for logging your traces in a more configurable way and has the following features:

  • Log String messages like trace(); but with log levels and restrict/skip level functionality.
  • Log all or specific properties of a (Display)Object.
  • Log all Stage children or below a specific Point(x,y).
  • Log with additional information like memory usage and/or timestamp.
  • Use it with your Actionscript (AIR/FP) project and listen to remote logging sessions (WiFi or wired) on Mac OSX or Windows.

Install

Before using LogX check for what purpose you will need it:

  • Download logx.swc to use inside your project as a library swc or download the nl.techtribe.logx classes.
  • For remote logging (outside your computer) download the AIR application to log through RTMFP.
  • To utilize your flashlog.txt you can markup your console in Eclipse with Grep Console : to color your flashlog-console inside Eclipse and check this to discover how to customize the colors.
